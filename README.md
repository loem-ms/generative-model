# Generative Model Note

### Run on your local PC
* Clone this repository
    ```
    $ git clone https://github.com/loem-ms/generative-model.git
    ```
* Setup
    - Python >= 3.10
    - Poetry: 1.8.2
    ```
    $ poetry install
    ```

* Build this book locally
    ```
    $ poetry run jupyter-book build ./
    ```
